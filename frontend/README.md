# AgentML Frontend

React + TypeScript web application for the AgentML platform.

## Tech Stack

- React 19
- TypeScript
- Vite
- React Router
- TailwindCSS

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run linting
npm run lint

# Run tests
npm test
```

## Pages

- **Projects** - Manage ML projects and workspaces
- **Data** - Upload files, view data sources, build training datasets with AI
- **Experiments** - Configure and run experiments with AI-powered setup
- **Models** - View trained models, metrics, feature importance
- **Visualize** - Generate charts with AI assistance
- **Settings** - Configure API keys and preferences

## Key Features

- **AI Setup Wizard** - 6-step pipeline for experiment configuration
- **Real-time Progress** - Live updates during training
- **Iteration Tracking** - Compare experiment iterations
- **Overfitting Monitor** - Visual warnings for model quality issues
- **What-If Analysis** - Interactive prediction exploration

## Configuration

The frontend connects to the backend API at `http://localhost:8000` by default.
